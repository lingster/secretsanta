## ğŸ“– How to Use

### For Organizers

1. **Open the App**: Navigate to the application URL
2. **Add Participants**: Enter names of all participants (minimum 2)
3. **Set Budget**: Enter the gift value/budget
4. **Choose Date**: Select the Secret Santa event date
5. **Generate**: Click "Generate Secret Santa Matches!"
6. **Share**: Copy the generated link and share it with ALL participants

### For Participants

1. **Open Link**: Click the link shared by the organizer
2. **Select Name**: Choose your name from the list
3. **Confirm**: Verify it's you (make sure no one is peeking!)
4. **View Assignment**: See who you're buying a gift for
5. **Keep Secret**: Don't tell anyone who you got!

## ğŸ”’ Privacy & Security

- **URL Encoding**: All participant data and matches are encoded in the URL
